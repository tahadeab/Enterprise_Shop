## 2. Design Style\n
- **Color Scheme**: Professional blue primary color (#2563EB) with white background, neutral gray tones (#F3F4F6, #6B7280) for secondary elements, green accent (#10B981) for success states, red (#EF4444) for alerts
- **Typography**: Clean sans-serif font family (Inter or similar), clear hierarchy with font sizes ranging from 14px to 32px, medium font weight (500) for headings
- **Layout Style**: Card-based design with subtle shadows (01px 3px rgba(0,0,0,0.1)), generous white space, 8px grid system for consistent spacing
- **Interactive Elements**: 6px border radius for buttons and cards, smooth transitions (200ms ease), hover states with slight elevation, primary buttons with solid fill, secondary buttons with outline style
- **Visual Hierarchy**: Clear separation between sections using background colors and borders, prominent call-to-action buttons, organized information architecture with logical grouping
- **Icons**: Consistent icon set (Heroicons or similar),20px-24px size for UI icons, outlined style for better clarity
- **Data Visualization**: Clean line charts and bar graphs using matching color palette, clear axis labels, interactive tooltips on hover
